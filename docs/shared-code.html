Département des outils d'aide

à la décision - SIES - MESR

Les bénéfices ques nous en retirons

  • Incitation à bien travailler
  • Travailler ensemble sur un même projet
  • Reconnaissance du travail effectué
  • Fluidifier les passations des compétences
  • Documentation et mémoire commune
  • Oblige à une réflexion sur la licence
  • ...

Workflow de travail

  • Relecture et validation du travail par un collègue via des PR
  • Implication des prestataires
  • 1 branche par environnement de travail (dev, staging, prod)
  • Utilisation des workflows (CI/CD)

Retour d'expérience

  • Ouverture plus difficile à faire dans un second temps
  • Le code n'est pas et ne sera jamais parfait
  • Outil technique qui ne se substituera jamais totalement à l'humain (réflexion, discussion...)
  • Adaptation rapide dans l'équipe même s'il subsiste une maîtrise de l'outil inégale en fonction de nos besoins et usages respectifs

Les points d'amélioration

  • Code mort / repo mort
  • Il reste des codes sources à ouvrir
  • Documentation
  • ...
